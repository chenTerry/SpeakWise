# AI 模拟对话应用 (AgentScope Dialogue Simulator)

## 项目概述

这是一个基于 **AgentScope 框架**的多智能体对话模拟平台，旨在为用户提供真实的、可定制的对话场景体验，帮助用户提升沟通技能、面试技巧和社交能力。

**产品定位**：通过 AI 模拟真实对话场景（面试、沙龙、项目会议、商务谈判等），提供安全的练习环境和即时反馈。

**目标用户**：求职者、职场新人、社交焦虑者、企业培训部门

## 技术架构

### 核心框架
- **AgentScope**: 分布式多智能体协作框架
- **后端**: Python + FastAPI
- **前端**: React + TypeScript（规划中）
- **数据库**: PostgreSQL + Redis + ChromaDB（规划）

### Agent 设计
```
┌───────────────────────────────────────────────────────┐
│                  AgentScope 框架                    │
├───────────────┬───────────────┬─────────────────────┤
│  Interviewer  │   Evaluator   │     Observer      │
│   Agent       │    Agent      │      Agent        │
├───────────────┼───────────────┼─────────────────────┤
│ • 提问策略引擎 │ • 实时评估引擎 │ • 对话状态跟踪    │
│ • 压力测试模块 │ • 多维度评分  │ • 情绪识别        │
│ • 反馈生成器   │ • 改进建议库  │ • 关键点标记      │
└───────────────┴───────────────┴─────────────────────┘
```

## 开发工作流

### 分支策略
```
main        ────●───────────●───────────●───>  # 稳定发布
              v0.1        v0.2        v1.0

develop     ────┼───┬───────┼───┬───────┼───>  # 开发主线
                │   │       │   │       │
feature/v0.1  ───┘   │       │   │       │
task/TASK-001 ──●───────>
```

### 常用命令
```bash
# 初始化项目
git init
git checkout -b develop

# 开始新版本
git checkout develop
git checkout -b feature/v0.1

# 开始任务
git checkout -b task/TASK-001-description

# 提交规范
git commit -m "feat(core): 创建项目结构 (TASK-001)"

# 版本发布
git tag -a v0.1 -m "Release v0.1 - 核心框架搭建"
```

### 提交类型
- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档变更
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建/工具变更

### 关键设计原则
1. **场景真实性**: 模拟真实职场对话的复杂性
2. **反馈专业性**: 提供可操作的改进建议（7 维度评估模型）
3. **成长可视化**: 用数据追踪用户能力提升

### 技术要点
- 使用 AgentScope 进行多 Agent 编排
- 支持多种大语言模型（DeepSeek、OpenAI 等）
- 实现上下文记忆和话题引导机制
- 构建可扩展的场景插件架构
